## Install the Docker packages
```
snap install docker
```
# Docker Command
## Build & Down Image
```
docker-compose down
docker-compose up -d --build
```

## Logs Image
```
docker logs -f {NAMES}
```


## List Image Active
```
docker ps
```

## List Image
```
docker images
```

## Stop & Remove Running Image
```
docker stop {Image NAMES}
docker rm {Image NAMES}
```
